## 🐛 Отладка уведомлений

После деплоя проверьте **логи Backend на Render**:

При запуске должно быть:
```
🔧 Инициализация Telegram бота для уведомлений...
TOKEN установлен: true
CHAT_IDS установлены: true
✅ Telegram bot инициализирован для отправки уведомлений
📋 Админов: 2
👥 Chat IDs: 123456789, 987654321
```

При создании тикета:
```
📤 Попытка отправить уведомление о новом тикете #1
📨 Отправка уведомления 2 админам...
✅ Уведомление отправлено админу: 123456789
✅ Уведомление отправлено админу: 987654321
```

Если видите:
```
⚠️ Telegram уведомления отключены
```

**Причины:**
1. `TELEGRAM_BOT_TOKEN` не установлен в backend
2. `TELEGRAM_ADMIN_CHAT_IDS` не установлены в backend
3. Chat IDs неправильные (проверьте через @userinfobot)

## 📋 Как получить Chat ID:

1. Напишите @userinfobot в Telegram
2. Бот пришлет ваш Chat ID (например: `123456789`)
3. Для нескольких админов разделите запятой: `123456789,987654321`
